# 文献计量学课程作业：人工智能在教育领域的应用研究

## 项目简介
本项目为文献计量学课程作业，围绕“人工智能在教育领域的应用研究”主题，完成文献检索、数据清洗与计量分析准备。

## 文件说明
- `检索方案.docx`：完整的文献检索策略与流程说明
- `csv.xlsx`：50 条清洗后的文献计量数据，包含标题、作者、年份、期刊、关键词、被引等字段
- `README.md`：项目整体介绍与使用说明

## 检索信息
- 检索数据库：中国知网（CNKI）
- 检索时间：2019-01-01 至 2025-12-31
- 检索日期：2026-03-25
- 最终数据量：50 条期刊论文

## 使用说明
1.  下载 `csv.xlsx` 可用于文献计量分析（如关键词共现、年度分布、期刊分布等）
2.  参考 `检索方案.docx` 可复现本次文献检索与数据清洗流程
3.  - `literature_analysis.py`：文献检索与数据清洗代码
- `code_README.md`：代码使用说明
